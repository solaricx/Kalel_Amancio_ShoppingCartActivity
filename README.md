# Shopping Cart System Quiz
Hello, this project is a simple console-based Shopping Cart System developed in C# and it will serve as the quiz requirement for our Computer Programming 2 subject.

# AI - Usage
I used AI as a guide while developing this system. It helped me understand how to use GitHub, particularly in managing and organizing my code. I also used AI to help me fix errors in my code and improve its logic, which allowed me to fix mistakes and make my program more efficient. But, I made sure to fully understand the code and personally edit and apply the necessary changes.

For enchancing the system, I also used it as a guide for updating and enchancing my system. It helped me develop the Category filter, View Cart System, Receipt generation with receipt number and date/time and Order history tracking. 

AI was also used in following ways:
- Suggest improvements for program structure and logic
- A guide to request a Pull Request in Github.

The final output reflects the my own understanding, and AI was used only as a guide and learning aid rather than a source of direct copying.
